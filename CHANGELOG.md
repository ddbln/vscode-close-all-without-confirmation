# Changelog

## [1.1.1] - 2026-02-20

### Changed
- Improved description

## [1.1.0] - 2026-02-20

### Changed
- Renamed to "Force Close All File Tabs" for better clarity
- Updated command titles and description

## [1.0.1] - 2026-02-19

### Changed
- Use "tabs" instead of "editors" in all user-facing text for better discoverability
- Updated description, README, settings, and quick pick labels

## [1.0.0] - 2026-02-19

### Added
- Close all editors (saved and unsaved) without repetitive confirmation dialogs
- Quick pick menu with options to close all editors or only editors in the active group
- Configurable `askBeforeClosing` setting (default: enabled)
- Configurable `activeGroupOnly` setting for direct-close mode
- Two commands: "Close All without Confirmation" and "Close All in Group without Confirmation"
- Ported from the popular [Sublime Text plugin](https://packagecontrol.io/packages/Close%20All%20without%20Confirmation) (3K+ installs)
